# Surely — Product Specification

## Overview
Parametric insurance protocol built on Chainlink CRE. Insurance pools with custom trigger conditions, multi-source data verification, AI adjudication, and instant on-chain payouts.

## Core Concepts
- **Insurance Pools**: Created by underwriters, each with custom trigger conditions
- **Policies**: NFT-based, purchased by policyholders
- **Triggers**: Automated conditions evaluated by Chainlink CRE workflows
- **Payouts**: Instant, on-chain, no claims process needed

## Architecture
- Smart contracts: Factory pattern for pool creation
- Frontend: Next.js with wallet integration
- Automation: Chainlink CRE for trigger monitoring and settlement
- Compliance: ACE (Autonomous Compliance Engine) integration

## Status
Early development — contracts scaffolded, webapp initialized.
